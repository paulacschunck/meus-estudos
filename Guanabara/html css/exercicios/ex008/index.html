Principais formatações

negrito/destaque

Nesa frase temos um termo em negrito usando a tag B(não semântica)

Nesta outra frase temos um termo em destaque usando a tag strong(semântica )

itálico/ênfase

Nesta frase temos um termo em itálico usando a tag i (não semântica)

Nessa frase temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

Podemos também um texto marcado usando a tag MARK

Texto grande e pequeno

Texto grande- tag BIG
Texto pequeno- tag Small

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser ido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. (semântica)

Existe a tag U que também insere uma linha abaixo da frase/palavra, mas sem semântica.

Texto sobrescrito

Para inserir coisas do tipo x24+3x=0

Texto subescrito

Para inserir coisas do tipo h2O
